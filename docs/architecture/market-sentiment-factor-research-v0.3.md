# 次日市场情绪预测 · 因子定义表 v0.3（讨论版）

> 研究性质文档。目标：通过纯技术面因子的频域共振分析，识别在何种市场状态下、哪些因子（或因子对）能与次日市场情绪强烈共振。
>
> 持仓周期目标：**7 天以内**，因子设计偏向"灵敏且具备短期持续性"。
>
> 研究输出形式：**报告**——回答"哪些因子在哪些状态下有意思"，不直接对接下游策略。

## 1. 研究目标

### 1.1 待预测对象 Y（复合次日情绪标签）

| 分量 | 业务含义 | 计算公式 |
|---|---|---|
| Y₁ | 次日市场平均涨幅 | `T+1` 日的流通市值加权 `pct_norm` 均值 |
| Y₂ | 次日上涨家数占比 | `T+1` 日 `count(pct>0) / N` |
| Y₃ | 次日涨跌停净广度 | `T+1` 日 `(limitUp - limitDown) / N` |
| **Y** | 复合情绪 | `α·Z(Y₁) + β·Z(Y₂) + γ·Z(Y₃)`，权重占位 `1:1:1`，Z 为过去 60 日 z-score 标准化 |

> 复合标签的权重在研究阶段保持等权占位；频域共振方法不依赖该权重的具体取值，而是看每个 Y 分量与因子的独立共振结构。

### 1.2 研究问题

1. **单因子条件共振**：因子 fᵢ 在哪些市场状态下，会在 7 天周期对应的频带（短频 F1、中短频 F2）上与 Y 强烈共振？
2. **因子对交叉共振**：两两因子组合（乘积/差/比）能否在某些状态下产生比单因子更强的共振？

### 1.3 持仓周期目标 → 频带优先级

| 频带 | 频率（cycle/day） | 周期（day） | 优先级 |
|---|---|---|---|
| F0 噪声 | >0.5 | <2 | 过滤 |
| **F1 短频** | 0.2-0.5 | 2-5 | **核心** |
| **F2 中短频** | 0.1-0.2 | 5-10 | **核心** |
| F3 中频 | 0.05-0.1 | 10-20 | 辅助 |
| F4 低频 | <0.05 | >20 | 趋势背景 |

## 2. 数据基础与预处理

### 2.1 数据来源

| 数据类型 | 来源 | 说明 |
|---|---|---|
| 个股日 K | `stock_daily_data` 表 | 已含前复权字段 `close_qfq` 等 |
| 股票基础信息 | `stock_info` 表 | 用 `market` 字段 + `ts_code` 后缀做板块识别 |
| **涨停明细** | **`tushare_limit_list_d` 表（已落库）** | 由 `LimitListDTable` 维护，研究阶段直接 SQL 读取，详见 §3 |
| 交易日历 | `tradingcalendar` 表 | 用于错位对齐 T 与 T+1 |

### 2.2 样本范围

- **起止时间**：2020-01-02 至最近交易日（约 1300+ 个交易日）
- **起点限制原因**：`tushare_limit_list_d` 表数据自 2020 年起
- **极端期处理**：2020-01-23 至 2020-04-30（疫情冲击期）单独标记，做共振分析时可选择降权或剔除观察

### 2.3 涨跌幅口径

```
pct_chg_raw  = close_qfq(T) / close_qfq(T-1) - 1
pct_norm     = pct_chg_raw / 个股预期涨跌幅上限 × 0.10
```

→ 所有股票统一压缩到 **±10% 等效坐标系**：主板原汁原味，创业板/科创板/北交所的实际收益被**压缩**进 ±10%，ST 股的实际收益被**放大**到 ±10%。这样跨板块加总、跨股票比较时的"满涨"权重一致。

### 2.4 个股预期涨跌幅上限识别

| 板块 / 类别 | 识别规则 | 预期上限 | 归一化系数（×0.10/上限） |
|---|---|---|---|
| 主板沪市 | `ts_code` 后缀 `.SH` 且代码以 `60` 开头 | 10% | × 1.00 |
| 主板深市 | `.SZ` 且代码以 `00` 开头 | 10% | × 1.00 |
| 创业板 | `.SZ` 且代码以 `30` 开头 | 20% | × 0.50 |
| 科创板 | `.SH` 且代码以 `688` 开头 | 20% | × 0.50 |
| 北交所 | `.BJ` | 30% | × 0.33 |
| ST 类 | 名称含 `ST` 或 `*ST`（任何板块） | 5% | × 2.00 |

> **判别优先级**：ST > 北交所 > 科创板 > 创业板 > 主板。例如名称带 ST 的创业板股票按 5%（×2.00）处理。
> **归一化后定义域**：`pct_norm ∈ [-0.10, +0.10]`（理论上极少越界，工程上 clip 到该区间）。
> **特殊日处理**：上市首日、新股开板首日、停牌复牌首日的实际涨跌幅可能远超上限，研究阶段一律剔除该交易日个股记录。

### 2.5 样本过滤

- 新股次新股：上市 ≤ 20 个交易日的剔除
- 停牌日：当日无成交量的剔除
- 退市过渡期：`delist_date` 前 30 个交易日剔除
- ST 股保留但归一化系数不同（见 §2.4）

### 2.6 加权约定

所有截面均值统一用 **流通市值（`mv_circ`）加权**，避免大量小盘股噪声主导信号。

### 2.7 平滑约定

所有"二阶差分类"因子统一采用 **EMA(3) 平滑后再做一阶差分**：

```
EMA(x, 3): α = 2/(3+1) = 0.5
           EMA(t) = 0.5·x(t) + 0.5·EMA(t-1)
```

## 3. 涨停数据使用（`tushare_limit_list_d` 直读）

### 3.1 表结构（已落库）

来源：`org.shiroumi.database.stock.table.LimitListDTable`，物理表名 **`tushare_limit_list_d`**。

| 字段 | 类型 | 因子用途 |
|---|---|---|
| `trade_date` | DATE | 主键之一 |
| `ts_code` | VARCHAR(15) | 主键之一 |
| `industry` | VARCHAR(64) | 辅助分组（暂不使用） |
| `name` | VARCHAR(64) | 辅助识别 |
| `close` | DOUBLE | 收盘价 |
| `pct_chg` | DOUBLE | 当日涨跌幅（原始非归一化） |
| `amount` | DOUBLE | 当日成交额 |
| `limit_amount` | DOUBLE | 板上成交额 |
| `float_mv` | DOUBLE | 流通市值 |
| `total_mv` | DOUBLE | 总市值 |
| `turnover_ratio` | DOUBLE | 换手率 |
| `fd_amount` | DOUBLE | 封单金额 |
| `first_time` / `last_time` | VARCHAR(16) | 首封 / 末封时间 |
| `open_times` | INT | **炸板次数**（C1 封板率的反向指标） |
| `up_stat` | VARCHAR(16) | 连板状态字符串如 `"5/7"`（C2/C3 来源） |
| `limit_times` | INT | 当日触及涨停次数 |
| `limit_type` | VARCHAR(2) | **U 涨停 / D 跌停 / Z 炸板**（C 组核心分类） |
| `updated_at_millis` | LONG | 落库时间，研究取数忽略 |

**复合主键**：`(trade_date, ts_code, limit_type)` —— 同一日同一股票可能同时出现 U 和 Z 记录（极少见，需注意去重）。

### 3.2 每日截面聚合 SQL 模板

```sql
SELECT
  trade_date,
  COUNT(*) FILTER (WHERE limit_type='U' AND open_times=0)         AS limit_up_clean,   -- 真·封板未开
  COUNT(*) FILTER (WHERE limit_type='U')                          AS limit_up_total,   -- 收盘涨停总数（含曾开板）
  COUNT(*) FILTER (WHERE limit_type='Z')                          AS flammed,          -- 炸板（未封住）
  COUNT(*) FILTER (WHERE limit_type IN ('U','Z'))                 AS triggered,        -- 当日曾触及涨停
  COUNT(*) FILTER (WHERE limit_type='D')                          AS limit_down,
  MAX(parse_up_stat(up_stat)) FILTER (WHERE limit_type='U')       AS consec_max,
  COUNT(*) FILTER (WHERE limit_type='U' AND parse_up_stat(up_stat) >= 2) AS consec_count
FROM tushare_limit_list_d
WHERE trade_date BETWEEN ? AND ?
GROUP BY trade_date;
```

> `parse_up_stat` 在 SQL 侧可用 `SUBSTRING(up_stat, 1, POSITION('/' IN up_stat)-1)::INT` 实现；落到 Repository 层时建议在 Kotlin 侧统一解析，避免方言差异。

### 3.3 `up_stat` 字段语义

格式 `"x/y"` —— **过去 y 个交易日内出现 x 次涨停**。

```
parse_up_stat("5/7") -> 5   // 连板高度近似值
```

注意：`up_stat` 是"窗口涨停频率"，不严格等于"连续涨停天数"。例如 `"5/7"` 可能是"涨涨涨跌涨涨涨"（断了一次又起）。研究阶段先用分子值近似 C2 连板高度，数据探索时与 `limit_type='U'` 的历史序列回算的"真·连板数"做对比，决定是否引入真连板字段。

## 4. 因子全表（36 个因子）

### 4.1 A 组：核心趋势与量价（14 个）

> 你原始诉求的 12 个 + 补齐市值中大盘档（A9a、A11a）

| ID | 业务名 | 计算公式 | 数据源 |
|---|---|---|---|
| **A1** | 市场平均涨幅 | `Σ(pct_norm_i · mv_circ_i) / Σmv_circ_i` | stock_daily_data |
| **A2** | 涨幅趋势加速度 | `Δ(EMA(A1, 3))` = `EMA(A1,3)(T) − EMA(A1,3)(T-1)` | A1 时序 |
| **A3** | 市场量变率 | `Σvolume_qfq(T) / Σvolume_qfq(T-1) − 1` | stock_daily_data.volume_qfq |
| **A4** | 量变趋势加速度 | `Δ(EMA(A3, 3))` | A3 时序 |
| **A5** | 涨停热度偏离 | `limitUp(T) − EMA(limitUp, 5)`，其中 `limitUp = limit_up_clean`（见 §3.2） | tushare_limit_list_d |
| **A6** | 跌停热度偏离 | `limitDown(T) − EMA(limitDown, 5)`（见 §3.2） | tushare_limit_list_d |
| **A7** | 小盘股涨跌幅 | T-1 日 `turnover_real ∈ [0, 10亿)` 的股票在 T 日的 `mean(pct_norm)`，加权 `0.6·T + 0.4·T-1` | stock_daily_data |
| **A8** | 中盘股涨跌幅 | T-1 日 `turnover_real ∈ [10亿, 50亿)` 同上 | 同上 |
| **A9a** | 中大盘股涨跌幅（新增） | T-1 日 `turnover_real ∈ [50亿, 100亿)` 同上 | 同上 |
| **A9b** | 大盘股涨跌幅 | T-1 日 `turnover_real ≥ 100亿` 同上 | 同上 |
| **A10** | 小盘股量变率 | T-1 日小盘股在 T 日 `mean(vol(T)/vol(T-1) − 1)`，加权同上 | 同上 |
| **A11** | 中盘股量变率 | T-1 日中盘股同上 | 同上 |
| **A11a** | 中大盘股量变率（新增） | T-1 日中大盘股同上 | 同上 |
| **A12** | 大盘股量变率 | T-1 日大盘股同上 | 同上 |

### 4.2 B 组：截面分布形状（6 个）

| ID | 业务名 | 计算公式 | 业务含义 |
|---|---|---|---|
| **B1** | 截面偏度 | `skew(pct_norm)` | 偏度正 = 少数大涨拉均值（虚假繁荣）；偏度负 = 少数大跌（情绪向恐慌倾斜） |
| **B3** | 分化度 | `std(pct_norm)` | 分化加剧通常是趋势转折前兆 |
| **B3'** | 分化度变化 | `B3(T) − EMA(B3, 3)` | 分化度的变化方向比绝对水平更重要 |
| **B4** | 上涨家数占比 | `count(pct > 0) / N` | 比均值更反映"广度"，权重股可能掩盖普跌 |
| **B5** | 强势股占比 | `count(pct_norm > 0.5) / N` | 即归一化涨幅 > 50% 涨停幅度（主板等价 >5%） |
| **B6** | 杀跌股占比 | `count(pct_norm < −0.5) / N` | 弱势广度 |
| **B7** | 广度持续性 | `Σ(B4(t) − 0.5) for t ∈ [T-2, T]` | 3 天累计的多空净广度，捕捉"持续偏强"vs"单日反弹" |

> **B2 截面峰度已删除**：v0.2 决定。对 7 天周期意义不大，且与 B3 高度相关。
> **共 7 个因子**（B1, B3, B3', B4, B5, B6, B7）

### 4.3 C 组：涨停质量与持续性（7 个）

> 7 天周期的核心组。tushare 升级后，C1/C7 从近似计算变为精确字段。

| ID | 业务名 | 计算公式 | 业务含义 |
|---|---|---|---|
| **C1** | 封板率 | `limitUp / triggered` = `count(limit_type='U' AND open_times=0) / count(limit_type IN ('U','Z'))` | 当日承接力，低 = 大面积炸板 |
| **C2** | 连板高度 | `MAX(parse_up_stat(up_stat))` | 龙头高度 = 情绪天花板 |
| **C2'** | 连板高度趋势 | `EMA(C2, 5)` | 高度的均值变化趋势 |
| **C3** | 连板梯队广度 | `count(parse_up_stat(up_stat) ≥ 2)` | 梯队厚度 = 情绪持续性 |
| **C4** | 近 3 日涨停股今表现 | T-1, T-2, T-3 日涨停股（去重）在 T 日的 `mean(pct_norm)` | 情绪延续性的核心指标 |
| **C5** | 涨跌停比 | `limitUp / max(limitDown, 1)` | 极端情绪方向 |
| **C6** | 连板赚钱效应 | T 日 `parse_up_stat ≥ 2` 的股票在 T 日的 `mean(pct_norm)` | 连板梯队是否还在赚钱 |
| **C7** | 断板率 | T-1 日涨停股（`limit_type='U'`）在 T 日**未涨停**的比例 | 情绪退潮最早信号 |

> **共 8 个因子**（C1, C2, C2', C3, C4, C5, C6, C7）

### 4.4 D 组：时序记忆与持续性（7 个）

> 窗口全部适配 7 天持仓周期：EMA20→EMA10，60 日分位数→20 日分位数。

| ID | 业务名 | 计算公式 | 业务含义 |
|---|---|---|---|
| **D1** | A1 短期 EMA | `EMA(A1, 5)` | 短期趋势锚 |
| **D2** | A1 中期 EMA | `EMA(A1, 10)`（**原 20 改 10**） | 中期趋势锚 |
| **D3** | 偏离中期均线 | `A1(T) − D2(T)` | 短期相对中期超买/超卖 |
| **D4** | A1 的 20 日分位数（**原 60 改 20**） | `rank(A1(T)) / 20` | 当前位置（相对意义） |
| **D5** | 量价背离信号 | `1 if sign(A1)·sign(A3)<0 else 0` | 量价不配合 = 反转征兆 |
| **D6** | 情绪状态持续天数 | `A1` 同号（正或负）的连续天数（带符号） | 7 天周期最直接的"持续性"度量 |
| **D7** | 动量衰减信号 | `1 if A1(T)<A1(T-1)<A1(T-2) AND all>0 else 0`（带对称负向） | 涨势/跌势在衰减 |

### 4.5 E 组：极值与波动（2 个）

| ID | 业务名 | 计算公式 | 业务含义 |
|---|---|---|---|
| **E1** | 截面振幅均值 | `mean((high-low)/prev_close)` 流通市值加权 | 全市场振幅放大 = 情绪剧烈 |
| **E2** | 振幅偏离 | `E1(T) − EMA(E1, 5)` | 振幅相对近 5 日的偏离 |

| 组 | 数量 | 主要刻画维度 |
|---|---|---|

## 6. 频域共振分析方法论（附录）

> 详细方案在 v0.2 已与用户对齐。本节作为研究输出形式的约束条件。

### 6.1 STFT 参数

- 窗长 **60 个交易日**，步长 **1 日**，Hann 窗
- FFT 点数 64（zero-padding 到 2 的幂）
- 频率分辨率 ≈ 1/64 cycle/day

### 6.2 相干分析（Magnitude-Squared Coherence）

```
γ²(t, ω) = |Sxy(t, ω)|² / (Sxx(t, ω) · Syy(t, ω))
```

- 每个因子 fᵢ 对 Y，输出 `γᵢ²(t, ω)`：时变频谱热力图
- 重点关注 F1（短频）+ F2（中短频）
- 显著性阈值：γ² > 0.5，且相位标准差 σ_φ < π/4

### 6.3 共振状态识别

用 3 个稳定描述因子作为"状态空间"，每个离散化 3 档：

| 状态维度 | 用哪个因子 | 三档划分 |
|---|---|---|
| 趋势方向 | D4（A1 20 日分位数） | <33% / 33-67% / >67% |
| 分化度 | B3（截面标准差） | 低三分位 / 中 / 高 |
| 量能水平 | EMA(A3, 5) | <0 缩量 / 平 / >0 放量 |

→ 3³ = 27 个状态格子；每个因子在每个格子里统计平均 γ²，识别"在哪个状态下共振"。

### 6.4 因子对的组合形式

候选组合（候选数 = C(38, 2) ≈ 700 对，用单因子 γ²_max > 0.3 剪枝到约 200 对）：

| 组合 | 公式 | 适用场景 |
|---|---|---|
| 乘积 | `fᵢ · fⱼ` | 两因子同向放大 |
| 差 | `fᵢ − fⱼ` | 两因子背离（如风格切换 A7-A12） |
| 比 | `fᵢ / fⱼ` | 相对强度（如量价比 A1/A3） |
