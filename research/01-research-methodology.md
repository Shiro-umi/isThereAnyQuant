# 短持仓因子频域共振研究方法学 v1.0

## 文档定位

本文档回答一件事：

> 在 7 天以内的短持仓场景下，如何判定一个技术面因子（或因子组合）在某个频域上，对某种未来走势具有稳定、领先、可兑现的预测贡献？

文档分为三部分：

1. **第一部分**讲研究目标和形式化框架，说明我们到底在问什么问题。
2. **第二部分**讲整套方法的设计原理，回答"为什么每一步要这么做"。
3. **第三部分**讲方法的有效性边界，说明在什么情况下结论可信、什么情况下应被怀疑。

具体的参数、公式、执行流程、输出字段定义，全部在配套的执行手册（`02-research-execution-handbook.md`）中。

---

## 第一部分　研究目标

### 1.1 研究对象与研究问题

研究对象是一组用于刻画 A 股短线市场情绪的技术面因子，包括：

- 38 个单因子，覆盖市场趋势、量价、风格、截面分布、市场广度、涨停生态、时序持续性、极值波动六大维度。
- 这些单因子经过筛选后构造的两两组合（差值型、乘积型、稳健比值型）。

研究**不**问：

> 哪个因子和未来走势的历史相关性最高？

而问：

> 哪个因子，在什么市场状态下，在哪个短周期频带上，对未来 1 至 5 天的哪类走势，有稳定且可验证的领先贡献？

形式化为：

```text
X(t) -> Y(t+h)

其中
  X(t) 为单因子或因子组合在 t 日的取值，
  Y(t+h) 为目标走势在未来 h 日的取值，
  h ∈ {1, 3, 5}。
```

### 1.2 走势的可计算定义

"未来走势"不是单一概念，至少要分四类：

| 走势类型 | 目标 Y | 业务含义 |
|---|---|---|
| 收益方向 | Y1 | 未来市场加权平均涨幅 |
| 市场广度 | Y2 | 未来上涨家数占比 |
| 极端情绪 | Y3 | 未来涨跌停净广度 |
| 复合情绪 | Y_composite | 上述三者的综合展示 |

研究主轴是 Y1 / Y2 / Y3 **分别**研究，Y_composite 仅作展示。原因是不同因子可能只对其中某一个分量有效，复合 Y 会掩盖因子的实际作用路径。

对短持仓而言，仅看未来均值不足以表达"短线兑现质量"，因此补充六个辅助标签：

| 标签 | 计算口径 | 表达 |
|---|---|---|
| Y_max_3 / Y_max_5 | 未来 3/5 日内最大 Y 值 | 短线冲高 |
| Y_min_3 / Y_min_5 | 未来 3/5 日内最小 Y 值 | 短线杀跌 |
| Y_drawdown_5 | 未来 5 日内的冲高回落幅度 | 短线兑现失败风险 |
| Y_direction_3 / Y_direction_5 | sign(Y_next_3/5) | 方向判断（分类目标） |

主研究产出仍以 `Y_next_1 / Y_next_3 / Y_next_5` 为核心，辅助标签用于评估"是否冲高""是否回落""是否退潮"。

### 1.3 频域的目标频带

研究只关心 2 至 10 天周期上的波动，划分为：

| 频带 | 周期 | 频率 (cycle/day) | 定位 |
|---|---:|---:|---|
| F1a | 2–3 天 | 0.333 – 0.500 | 极短线反抽 / 退潮 |
| F1b | 3–5 天 | 0.200 – 0.333 | 短线主交易频带 |
| F2a | 5–8 天 | 0.125 – 0.200 | 7 天内延续核心频带 |
| F2b | 8–10 天 | 0.100 – 0.125 | 辅助观察 |

主排名只使用 **F1b / F2a**，F1a 作为二级核心，F2b 仅辅助观察。10 天以上的频带（F3、F4）只能作为"市场背景状态"使用，不参与短线因子排名。

这是相对早期版本 F1 (2–5)、F2 (5–10) 的细分。**理由见 §2.2**。

### 1.4 形式化研究问题

把走势、频带、horizon、状态四个维度组合起来，整套研究要回答的形式化问题是：

```text
对于因子 X，
存在频带 B ∈ {F1a, F1b, F2a, F2b}，
存在 horizon h ∈ {1, 3, 5}，
存在 Y 分量 Y_k ∈ {Y1, Y2, Y3, ...辅助标签},
存在市场状态 s ∈ S,
使得在状态 s 下，X 在频带 B 上对 Y_k(t+h) 具有稳定、领先、样本外可兑现的预测贡献。
```

这个问题的答案不是 "是 / 否"，而是一张多维卡片（见执行手册 §共振卡片）。

## 第二部分　方法设计原理

### 2.1 为什么必须先拆频，再谈相关性

直接计算 `corr(X, Y)` 会把所有周期成分混在一起：日内噪声、3 天波动、20 天趋势、季度因素都进了同一个数。结果是：

- 高 `corr(X, Y)` 可能来自慢周期的同步漂移（X 和 Y 都在缓慢上涨），但短线交易关心的是 3–5 天频段的波动是否一致。
- 慢周期的强同步会掩盖快周期的反向关系，反之亦然。

对短持仓研究而言，先把 X 和 Y 各自分解到目标频带 B，再研究：

```text
X_B(t) -> Y_B(t+h)
```

才能保证我们看到的"相关"是**目标周期上的相关**，而不是不同周期的混合。

带通滤波 + 滚动相关只是"快速发现"，**频域确认要用 STFT 相干性**，原因见 §2.4。

### 2.2 为什么把 F1/F2 进一步细分

早期方案是 F1 = 2–5 天、F2 = 5–10 天。对 7 天以内持仓这个目标，F2 = 5–10 天偏宽：

- 10 天周期已经超出主要持仓窗口，容易混入"中短趋势背景"。
- 5 天和 8 天的因子行为往往不同（5 天更接近一周交易节奏，8 天靠近"半月度"波段），合在一起会让相干性指标平均化、稀释。

细分到 F1a / F1b / F2a / F2b 之后：

- F1b（3–5 天）和 F2a（5–8 天）恰好覆盖 7 天以内持仓最相关的两个交易节奏。
- F1a（2–3 天）单独抽出来研究极短线反抽与退潮，避免被 F1b 平均掉。
- F2b（8–10 天）保留作为"接近中短趋势的辅助观察带"，不进主排名。

这样划分有一个统计学副作用：每个频带的频率宽度变窄，STFT 在该频带内的频点数减少，相干性估计的方差变大。我们通过两个手段对冲：

1. STFT 频带聚合时按频点数加权平均（见执行手册 §STFT 频带聚合）。
2. 单频带样本不足时合并到相邻频带（F1a 不足合 F1b，F2b 不足合 F2a），合并规则在执行手册中。

### 2.3 为什么主目标 Y 要拆成 Y1 / Y2 / Y3

复合 Y 会掩盖因子的作用路径，举三个例子：

- 涨停股赚钱效应（C4）影响的是**广度扩散**，体现在 Y2，但对 Y1（加权平均涨幅）影响微弱（因为涨停股本身权重不高）。
- 大盘股动量（A9b）影响 Y1，但几乎不改变 Y2（少数大票拉指数不代表多数股票上涨）。
- 断板率（C7）影响 Y3（极端情绪退潮），但 Y1 和 Y2 在短期内可能还没变化。

如果只看 Y_composite，上述三种因子的频域共振信号会被互相平均、相互掩盖。

拆分 Y 之后：

- 每个因子的卡片会显式标注"对哪个 Y 分量有效"。
- 因子组合的设计有了明确目标（差值型适合广度 vs 涨幅的背离，乘积型适合广度 × 极端情绪的共振）。
- A 类有效因子的数量天然会上升（因为粒度细化），但每张卡片的信息密度提高。

### 2.4 为什么频域确认必须用 STFT 相干性

带通滤波 + 滚动相关是**时域**方法：

- 优点：直观，能直接看到"X 领先 Y 几天"。
- 缺点：滤波器的相位响应会污染领先关系（零相位滤波泄露未来信息，因果滤波引入延迟），并且滚动相关无法分辨"两个序列在 4 天频率同步"和"两个序列在 5 天频率同步"。

STFT 相干性 γ²(t, ω) 是**时频**方法：

- 在每个时间窗口 t 内，给出每个频率 ω 上 X 和 Y 的同步强度（取值 0–1）。
- 同时给出相位差 φ(t, ω)，可换算出领先天数 `lead_days = φ / (2πω)`。
- 不依赖单一滤波器的相位响应，受滤波器形式影响小。

两种方法的合理分工：

| 步骤 | 工具 | 用途 |
|---|---|---|
| 第一层：发现 | 带通滤波 + 滚动相关 | 快速筛掉"明显无效"的因子，得到候选池 |
| 第二层：确认 | STFT 相干性 + 相位 | 在候选池上做频域稳健确认 |
| 第三层：验证 | 样本外 IC / HitRate / Top-bottom spread | 判断频域共振是否在样本外兑现 |

第一层是**召回优先**（宁可多收，不漏报），第二层是**精度优先**（剔除时域相关但频域不稳定的因子），第三层是**业务落地优先**（频域共振不等于可交易）。

### 2.5 为什么"领先天数"是硬约束

这是把"频域相关现象"和"短线有效因子"分开的关键。

一个因子可以满足"高 coherence + 高 coverage + 相位稳定"但**完全不能预测未来**——只要它和 Y 是同步的（lead_days ≈ 0）或滞后的（lead_days < 0）。这种因子在频谱上看起来很漂亮，但拿去做交易决策会发现"信号出现时 Y 已经走完了"。

因此我们规定：

| horizon | 有效领先区间（天） |
|---|---:|
| h = 1 | 0.5 – 1.5 |
| h = 3 | 1 – 3 |
| h = 5 | 1 – 5 |

落在区间内才能进 **A 类（短线强发现）**；同步的因子降级为 **C 类（解释型）**；滞后的因子直接 Reject。

实际计算领先天数有两种方法（相位换算、时域 lag correlation），它们在噪声大的短频段经常给出不一致结果。本研究的裁决规则是：

```text
以时域 lag correlation 的峰值位置为准（更鲁棒）。
相位换算作为一致性交叉验证。
当两者差距 > 1 天时，标记该因子的领先关系为"不稳定"，降级为 B 类。
```

### 2.6 为什么必须做样本外验证

样本内的所有指标（包括 STFT 相干性、滚动相关、状态内 IC）都受到**回看偏差**和**多重检验偏差**污染：

- 回看偏差：我们用 t 时刻已知 Y(t+h) 之后才决定研究哪些因子、哪些频带，导致结论自带"上帝视角"。
- 多重检验偏差：38 因子 × 4 频带 × 3 horizon × 4 Y 分量 × 27 状态 ≈ 49000 个测试单元，按 5% 显著性水平随机也会产生约 2500 个"显著"结果。

样本外验证用滚动窗口 walk-forward：

```text
训练窗口 500 日（用于选因子、定参数、估计阈值）
验证窗口 120 日（在训练窗口之后，纯前向预测）
滚动步长 20 日（窗口前移）
```

这种结构保证：

- 每一次"选择"只用过去信息。
- 每一次"评估"只在未来窗口内进行。
- 通过滚动覆盖整个样本，得到的 OOS 指标分布反映了"在不同市场环境下方法是否稳健"。

进 A 类的硬指标必须包含 `OOS_IC > 0`、`HitRate > baseline`、`Top-bottom spread 稳定为正`，否则即使样本内结果再漂亮也不能采用。

### 2.7 为什么状态条件分析是必要的

短线因子的有效性**强烈依赖市场状态**。三类常见情形：

- 涨停股赚钱效应（C4）在"中位 + 低分化 + 放量"状态下对 Y2 有强延续作用，但在"低位 + 高分化"状态下完全失效甚至反向。
- 断板率（C7）在"高位 + 高分化 + 放量"状态下对 Y3 有强退潮预警，但在低位启动初期可能是健康换手的表现。
- 小盘 / 大盘风格因子（A7 - A9b）在"风格切换"状态下有强信号，但在风格稳定期是噪声。

如果不分状态，因子的"全局有效性"会被各状态相互抵消，最终得到"统计上轻微显著但实际无法使用"的结果。

状态采用三维网格：

| 维度 | 分档 |
|---|---|
| 趋势方向（D4 或 A1 近 20 日分位） | 低位 / 中位 / 高位 |
| 分化度（B3 或 B3'） | 低 / 中 / 高 |
| 量能水平（EMA(A3, 5)） | 缩量 / 平量 / 放量 |

形成 3 × 3 × 3 = 27 个状态。

状态窗口的选择对短持仓研究是关键：

| 状态窗口 | 用途 | 优先级 |
|---|---|---|
| 20 日 | 快速切换捕捉，噪声大 | 辅助 |
| 60 日 | **短持仓主状态** | 主用 |
| 252 日 | 长期背景，不直接决定短线排名 | 背景 |

60 日状态是短持仓研究的主窗口，原因是 60 日恰好覆盖约一个季度，能反映"当前市场处于哪种阶段"，又不会像 252 日那样过于平滑。

### 2.8 为什么因子对要做增量贡献检验

38 个因子两两组合理论上有 703 对，每对可以做差值、乘积、稳健比值，总计 2109 个组合。盲目全搜索的问题：

- 大量组合是冗余的（因为单因子之间本身就高度相关）。
- 多重检验的代价巨大（FDR 修正后能进 A 类的极少）。
- 即使某个组合的样本内表现优于单因子，也可能只是"过拟合搜索"的产物。

因此因子对必须做**增量贡献检验**：

```text
基础模型:  Y_B(t+h) = α + β1·Zi_B(t) + β2·Zj_B(t) + ε
加入组合:  Y_B(t+h) = α + β1·Zi_B(t) + β2·Zj_B(t) + β3·Gij_B(t) + ε
```

只有当 β3 显著为正、并且 OOS IC / Top-bottom spread 相对基础模型有提升时，因子对才被认为"提供了独立信息"。

**增量贡献的基线（base）的具体定义**在执行手册 §因子对评估中说明，研究规定使用"当前已入选 A/B 类因子的 PCA 前 5 个主成分"，避免同族因子互相 cannibalize 增量。

### 2.9 为什么显著性检验要用 Block Permutation

金融时间序列有强自相关，常规随机重排（permutation by day）会破坏自相关结构，让零假设分布过于狭窄，导致 p 值人为偏低，假阳性激增。

Block permutation 保留**块内**自相关，只打乱**块间**顺序：

```text
1. 按 block_size 天切块
2. 随机打乱块顺序
3. 重新计算 Score
4. 重复 N 次，得到 H0 分布
5. 计算真实结果在 H0 分布上的分位
```

block_size 的选择必须和频带的特征长度匹配：

- F1a（2–3 天）：block_size = 5 天
- F1b（3–5 天）：block_size = 10 天
- F2a（5–8 天）：block_size = 20 天
- F2b（8–10 天）：block_size = 20 天

如果 block_size 远大于频带周期，会保留太多周期结构，让 H0 分布偏向高 coherence，反而让显著性检验**过于宽松**。统一用 20 天 block 是错的（早期方案的不足）。

多重检验上使用 FDR Benjamini-Hochberg，q < 0.1 作为探索性阈值。

### 2.10 双版本预处理（敏感 + 稳健）

每个因子计算两个标准化版本：

- **Z_60**：60 日滚动 z-score。短线敏感版本，捕捉近期变化。
- **Z_120**：120 日滚动 z-score。稳健版本，平滑短期噪声。

每个版本再衍生一个去趋势版本 `Z_*_detrend(t) = Z_*(t) - EMA(Z_*, 20)`。

主发现优先级：

```text
Z_60_detrend > Z_60 > Z_120_detrend > Z_120
```

逻辑是：短持仓研究优先用敏感版本去发现"近期生效"的因子，再用稳健版本去验证"是不是只是短期噪声"。如果一个因子在 Z_60_detrend 上显著但在 Z_120 上完全不显著，要怀疑它是噪声驱动；如果两个版本都显著，可信度大幅提升。

## 第三部分　有效性边界与方法局限

### 3.1 频域相干不等于预测力

这是整套方法最容易被误读的点。

高 coherence 只能说明：

> X 和 Y 在该频带上的波动是同步的。

它**不能**说明：

- 这种同步是因果的。
- 这种同步能在样本外延续。
- X 出现某个值时 Y 一定朝某个方向走。

要建立"预测"关系，必须同时验证：

1. 因子领先 Y（相位 / lag correlation）。
2. 带通回归系数 β 方向稳定且有业务解释。
3. 样本外 OOS IC、HitRate、Top-bottom spread 都为正且稳定。
4. Block permutation 显著性 + FDR 修正通过。
5. 状态内样本足够（≥ 60）。

只满足"高 coherence"的因子，结论上只能称为：

> 频域相关现象。

不能称为：

> 短线有效因子。

### 3.2 60 日 STFT 单窗口的滞后风险

STFT 窗口长度决定了频域分辨率和时间分辨率的权衡：

- 长窗口（60 日）：频率分辨率高，但对"近期变化"反应慢，可能错过 5–10 天内的快速切换。
- 短窗口（30 日）：时间分辨率高，但频率分辨率差，F1a / F1b 频带的相干性估计误差大。

解决方案是**多窗口 STFT 交叉验证**：

| 窗口 | 定位 |
|---|---|
| 30 日 | 短线敏感确认 |
| 40 日 | **主确认窗口** |
| 60 日 | 稳健确认 |

主结论以 STFT-40 为准，STFT-30 / 60 用作敏感性 / 稳健性的左右校验。一个因子如果只在 STFT-30 显著但 STFT-40 / 60 都不显著，要降级处理。

### 3.3 状态过细导致样本不足

27 状态 × 250 个交易日/年 × 4 年样本 ≈ 27000 个观测，平均每个状态约 1000 个观测。但状态分布**极度不均**：

- "中位 + 中分化 + 平量"可能占 20% 以上。
- "低位 + 低分化 + 放量"可能不到 1%。

样本不足的状态结论必须谨慎：

- 状态内有效窗口数 < 30：不输出结论。
- 30 ≤ N < 60：仅作为"观察"，不进 A 类。
- N ≥ 60：可参与排序。

样本不足时的合并规则采用**层级化合并**（执行手册中给出具体顺序）：

```text
第一优先：合并分化度相邻档（低↔中、中↔高）
第二优先：合并量能相邻档（缩量↔平量、平量↔放量）
最后：合并趋势相邻档（低位↔中位、中位↔高位）
```

合并的优先级反映了维度的"可塑性"：分化度档位之间的差异最连续，趋势档位之间的差异最显著，所以优先合分化、最后合趋势。

### 3.4 极端行情下的方法稳健性

A 股有几段公认的极端期：

- 2020-01-23 至 2020-04-30：疫情冲击。
- 2022 全年的系统性下跌阶段。
- 2024-09 之后的快速大涨阶段。
- 任何千股涨停 / 千股跌停的极端交易日。

这些时期会让因子的统计性质（均值、方差、自相关）大幅偏移。处理方式不是简单删除，而是**三口径并跑**：

| 口径 | 用途 |
|---|---|
| 全样本 | 总体表现 |
| 剔除极端期 | 常规市场表现 |
| 极端期单独分析 | 极端状态下的特殊因子 |

最终结论标注：

- "常规有效" / "极端期有效" / "只在极端期有效" / "极端期失效"

这种标注让用户知道结论的适用范围，而不是给一个"全样本平均有效"的误导性结论。

### 3.5 因子对的过拟合风险

因子对的搜索空间天然比单因子大一个数量级，过拟合风险更高。三类常见过拟合：

- **比值型过拟合**：当分母接近零时数值爆炸，少数极端点主导整个相关性。解决：使用 `Zi / (|Zj| + ε)` 而不是 `Zi / Zj`，ε 取 0.5 或 1.0。
- **同族因子互相代理**：A1（市场涨幅）和 D1（涨幅 EMA）本质上是同一信息，组合 A1 - D1 看起来"独立"但提供不了新信息。增量贡献检验的 base 用 PCA 主成分即是为此设计。
- **状态切片过细**：因子对在某个小状态内"显著"，但状态内只有 30 个观测。状态内样本数门槛对因子对要更严格（建议 ≥ 80）。

### 3.6 滤波器选择的因果性问题

带通滤波在 Python 里有两种常见调用：

- `filtfilt`：零相位滤波，前后两次正反向滤波。**会泄露未来信息**。
- `lfilter`：单向因果滤波，引入相位延迟但不泄露未来。

样本内发现阶段可以用 `filtfilt`（追求频率精度，不假装预测），但样本外验证阶段**必须**用 `lfilter` 或显式截断 `filtfilt` 的前向溢出（前向延迟 = 滤波器阶数）。

研究规定：

```text
带通滤波器：Butterworth 4 阶
样本内发现：filtfilt（双向，零相位）
样本外验证：lfilter（单向因果）
两种结果都要保存，对比"是否因为未来函数才显著"。
```

如果一个因子在 `filtfilt` 下显著但在 `lfilter` 下消失，说明它在样本内"看到了未来"，必须 Reject。

### 3.7 状态切换本身没作为信号源（已知盲区）

本方法把市场状态作为**条件变量**：在状态 s 下，因子 X 是否对 Y 有效。

但**状态切换本身**——从"低位高分化"切到"中位低分化"——可能比静态状态包含更多信号。当前研究没有覆盖这个维度，是已知盲区。

可能的扩展方向（不属于 v1.0 范围）：

- 把"状态切换事件"作为新的目标 Y。
- 把"距上一次状态切换的天数"作为新的因子。
- 研究因子在"状态切换前 N 日"的行为变化。

## 第四部分　最终判定原则

一个因子或因子对要被认定为"短持仓有效发现（A 类）"，必须**同时**满足以下条件：

```text
1. 因子和 Y 都被拆到同一个目标频带（F1b 或 F2a）。
2. STFT 相干性 MeanCoh > 0.5。
3. 高相干覆盖率 Coverage > 30%。
4. 相位稳定 PhaseStd < π/4。
5. 因子领先 Y，领先天数落在 horizon 对应的有效区间内。
6. 带通回归系数 β 方向稳定且业务可解释。
7. 状态内样本数 ≥ 60。
8. 样本外 OOS IC > 0.05，HitRate 高于基准，Top-bottom spread 稳定为正。
9. Block permutation 显著性 + FDR 修正（q < 0.1）通过。
10. 对同类因子（PCA 主成分）有增量贡献。
11. filtfilt 和 lfilter 两种因果性下结论一致。
```

只满足部分条件的因子分为：

- **B 类（观察因子）**：频域结果较强，但样本外或状态依赖性较强，或样本数不足。
- **C 类（解释型因子）**：高相干但不领先 Y，或仅同步反映市场状态。可用于解释市场，不作预测。
- **Reject**：相位不稳定 / 只在少数窗口有效 / 样本外失效 / 显著性不过 / filtfilt 与 lfilter 结论不一致。

最终的结论表达必须包含**五个维度**：

```text
因子: <name>
频带: <F1b / F2a / ...>
走势: <Y1 / Y2 / Y3 / 辅助标签>
horizon: <h=1 / 3 / 5>
状态: <趋势 × 分化 × 量能>
```

任何缺少其中任一维度的结论都是不完整的。例如：

不要写：

> C4 对市场有强影响。

要写：

> C4 在 F2a (5–8 天频带) 上，对 Y2_next_3 (未来 3 日上涨家数占比) 在"趋势中位 + 低分化 + 放量"状态下有强影响。证据：MeanCoh = 0.58，Coverage = 36%，PhaseStd = 0.62，领先 2.1 天，OOS IC = 0.08，HitRate = 0.61，q = 0.04。业务解释：涨停股赚钱效应在该状态下向市场广度扩散，对未来 3 日情绪延续有预测贡献。

## 附录　核心方法论一句话总结

```text
第一，先拆频：只研究因子和走势在同一个短周期频带上的关系。

第二，看领先：只有因子频带成分领先走势频带成分，才可能有预测意义。

第三，看兑现：频域共振必须在样本外转化为 IC、命中率或高低分组差异。

第四，看状态：短线因子不是全局有效，必须说明它在哪种市场状态下有效。
```

研究的最终产出不是"因子排行榜"，而是一张张可被业务理解的**共振卡片**——每张卡片说明因子、频带、走势、horizon、状态、证据、解释，缺一不可。
